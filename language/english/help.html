Summary

  • This module provides an advanced counter and statistics for Xoops.

Install

  • For install correctly the module, you do not need to modify any source code.
  • Install the block "collector" and give the right access to it for all defined group, you must define on which page you want to collect user visit (information). If you want to collect on all site page, please selct "all page".
  • Install block "counter" by editing it for select all options that you need. This block just display the result collected by "collector" block. If you have an old counter and you want to add this old value, you can add it and this value will be automatically add on counter visit but not on the statistics.
