Présentation

  • Ce module est outil compteur avancé pour Xoops.

Installation

  • Aucun modification particulière des fichiers n'est nécessaire pour installer Xoops*Visit.
  • Activer le bloc "Collector" et distribuer les accès selon les groupes à comptabiliser. Indiquez sur quelle(s) page(s) les informations visiteurs doivent être collectées (sélectionnez "Toutes les pages" si vous souhaitez enregistrer toutes les visites)
  • Activer le bloc "Countter" et paramétrez ses options. Ce bloc affiche les informations collectées. Si vous disposiez d'un ancien compteur de visites, vous pouvez saisir un chiffre initial qui sera automatiquement incrémenté.
